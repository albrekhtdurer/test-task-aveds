# Тестовое задание Aveds

## Описание

Проект выполнен на `React + Typescript`, сборка - `Vite`. Для маршрутизации использован `react-router-dom`.

Настроена маршрутизация: если пользователь авторизован (данные об авторизации получаем из localStorage), по кнопке "Войти" переход сразу на страницу пользователя.

## Запуск

Для запуска проекта:

1. Установить зависимости `npm i`.
2. Запустить проект `npm run dev`
3. Проект будет доступен по адресу `http://localhost:5173/`
